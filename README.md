# SOC Analyst Lab Portfolio

This repository contains hands-on SOC Analyst Level 1 labs completed using the TryHackMe SOC Level 1 learning path.

The focus of this portfolio is on:
- Security alert triage
- Log analysis
- Phishing investigation
- Incident documentation and reporting

Each lab documents the investigation workflow, findings, and recommendations, simulating real-world SOC analyst tasks.

## Structure
- Each folder represents a completed SOC investigation
- Screenshots are included to demonstrate analysis steps
- A reusable SOC investigation template is provided

## Tools & Skills Practiced
- SIEM analysis
- Log review
- IOC identification
- Incident classification
- Security reporting
